Exploratory Data Analysis on Netflix Dataset
📖 Project Description

This project performs Exploratory Data Analysis (EDA) on the Netflix dataset using Python. The analysis helps understand the structure of the dataset and discover useful insights such as content type distribution, top countries, and release year trends.

🛠 Technologies Used
Python
Pandas
Matplotlib
Google Colab
📂 Dataset

Dataset Used: Netflix Titles Dataset

Columns include:

Title
Type
Country
Release Year
Rating
Genre
📊 Analysis Performed
Dataset inspection
Missing value checking
Movies vs TV Shows analysis
Top countries analysis
Release year analysis
Basic data insights
🔍 Insights
Movies are more common than TV Shows on Netflix.
USA contributes the highest amount of Netflix content.
Netflix content increased significantly after 2015.
▶️ How to Run
Open Google Colab
Upload the dataset file
Run the Python code cells
View outputs and insights
